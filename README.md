# traveller-sheet-KR

트래블러 Mongoose 2e Foundry VTT 한글 캐릭터 시트 모듈

## 설치

Foundry VTT → Add-on Modules → Install Module에서 manifest URL 입력:
```
https://github.com/foredayner/traveller-sheet-KR/releases/latest/download/module.json
```

## 요구사항

- Foundry VTT v13+
- mgt2e 시스템 (Mongoose Traveller 2e) v0.19+
- **traveller-items-KR** (필수)

## 주요 기능

- 한글 캐릭터 시트 (정보/기능/장비/상점/경력/배경 탭)
- 장착 아이템 특성치 자동 적용 (STR/DEX/END/INT/EDU/SOC)
- 방어구 요구기능 미달 시 전판정 패널티 자동 반영
- 컴퓨터 내장 장착 시 전자기기(컴퓨터) 기능 자동 부여
- 굴림 다이얼로그에 보유 아이템 보너스 자동 표시
- 한글 아이템 시트 (무기 특성 태그 UI 포함)
- traveller-items-KR 컴펜디엄 연동
